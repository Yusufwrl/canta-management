## 🎯 Turhost cPanel Adımları

### Adım 1: cPanel'e Giriş
1. Turhost hesabından cPanel'e gir
2. File Manager'ı aç
3. public_html klasörüne git

### Adım 2: Dosya Yükleme
1. "Upload" butonuna tıkla
2. Yukarıdaki dosyaları/klasörleri seç
3. Yükleme tamamlanana kadar bekle

### Adım 3: Node.js Ayarları
1. cPanel'de "Node.js" sekmesine git
2. "Create Application" tıkla
3. Ayarlar:
   - Node.js Version: 18+
   - Application Root: public_html
   - Application URL: yourdomain.com
   - Startup File: server.js

### Adım 4: Database Kurulumu
1. "PostgreSQL Databases" aç
2. Database oluştur: canta_management
3. Kullanıcı oluştur ve şifre belirle
4. .env dosyasında DATABASE_URL güncelle
